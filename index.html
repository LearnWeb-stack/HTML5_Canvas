01
requestAnimationFrame
window.requestAnimationFrame( callback )

What is it? A browser API that says: "Before you paint the next screen frame, run my function." It fires roughly 60 times per second, in sync with your monitor's refresh rate.

Why not setInterval? setInterval runs on a timer regardless of what the screen is doing — it can fire too early, too late, or pile up missed frames. requestAnimationFrame is always perfectly in sync with the display, and it automatically pauses when the tab is hidden, saving CPU and battery.

The animation loop pattern You create a loop by having your function call requestAnimationFrame(itself) at the end. Each call schedules exactly one upcoming frame. It returns a numeric ID you can pass to cancelAnimationFrame(id) to stop the loop.

let x          = 0;
let direction  = 1;
let frameId    = null;
let frameCount = 0;
let running    = false;

function animate() {
  x += 3 * direction;
  if (x > 280 || x < 0) direction *= -1;

  document.getElementById('box').style.left =
    x + 'px';

  document.getElementById('counter').textContent =
    'Frame: ' + (++frameCount);

  frameId = requestAnimationFrame(animate);
}

document.getElementById('btn').onclick = function() {
  if (running) {
    cancelAnimationFrame(frameId);
    this.textContent = 'Start';
  } else {
    animate();
    this.textContent = 'Pause';
  }
  running = !running;
};

02
getBoundingClientRect
element.getBoundingClientRect()

What is it? A method available on any HTML element that returns a DOMRect object containing the element's exact size and position — measured in pixels, relative to the viewport (the visible area of the browser window).

It always reflects the element's current rendered state, so it works correctly even after CSS transforms, scrolling, or layout changes.

PropertyWhat it tells you
rect.top Pixels from the viewport top to the element's top edge
rect.left Pixels from the viewport left to the element's left edge
rect.right Pixels from the viewport left to the element's right edge
rect.bottomPixels from the viewport top to the element's bottom edge
rect.width The element's rendered width in pixels
rect.heightThe element's rendered height in pixels

03
Collision Detection
requestAnimationFrame + getBoundingClientRect combined

How does it work? On every animation frame, we call getBoundingClientRect() on two elements and compare their edges. Two rectangles are not overlapping if one is completely outside the other on any axis. We check for all four "no collision" cases and negate them.

The collision formula — AABB (Axis-Aligned Bounding Box) Two rectangles A and B are colliding when all four of these are true at once:

  A.right > B.left   ·   A.left < B.right
  A.bottom > B.top    ·   A.top < B.bottom

Equivalently: they are not colliding when A is fully left, right, above, or below B. Negate those four conditions with ! and you get the collision check.

const player = document.getElementById('player');
const wall   = document.getElementById('wall');
const status = document.getElementById('status');
let x = 0, dir = 1, running = false, id;

function isColliding(a, b) {
  const r1 = a.getBoundingClientRect();
  const r2 = b.getBoundingClientRect();
  return !(
    r1.right  < r2.left   ||
    r1.left   > r2.right  ||
    r1.bottom < r2.top    ||
    r1.top    > r2.bottom
  );
}

function loop() {
  x += 2 * dir;
  if (x > 340 || x < 0) dir *= -1;
  player.style.left = x + 'px';

  if (isColliding(player, wall)) {
    player.style.background = '#f06060';
    wall.style.background   = '#f06060';
    status.textContent      = 'COLLISION!';
  } else {
    player.style.background = '#4ec990';
    wall.style.background   = '#555';
    status.textContent      = 'No collision';
  }

  id = requestAnimationFrame(loop);
}

document.getElementById('btn').onclick = function() {
  running = !running;
  running ? loop() : cancelAnimationFrame(id);
  this.textContent = running ? 'Pause' : 'Start';
};

04
Jump on Collision
collision → velY + gravity loop

What's the idea? When isColliding() returns true, we set velY to a negative number (upward force). A gravity value is added to velY every frame, gradually pulling the square back down to the ground.

Three new variables are added on top of Section 03: y (height above ground), velY (current vertical speed), and hasJumped — a one-shot lock that prevents the jump re-firing every frame while the boxes are still overlapping.

Why hasJumped matters Without it, isColliding() stays true for several frames while boxes overlap — resetting velY every frame cancels the jump before it travels anywhere. Setting hasJumped = true on the first hit locks out further triggers. It's reset to false only when the square lands back on the ground (y <= 0).

// ── State ──────────────────────────────────────
let x = 20,  velX = 2;
let y = 0,   velY = 0;

const GRAVITY    = 0.6;
const JUMP_FORCE = -12;     // negative = upward
const GROUND     = 0;
let   hasJumped  = false;   // one-shot lock
let   running = false, id;

// ── Collision (same as Section 03) ─────────────
function isColliding(a, b) {
  const r1 = a.getBoundingClientRect();
  const r2 = b.getBoundingClientRect();
  return !(r1.right  < r2.left  || r1.left > r2.right ||
            r1.bottom < r2.top   || r1.top  > r2.bottom);
}

// ── Loop ───────────────────────────────────────
function loop() {
  const player = document.getElementById('player');
  const wall   = document.getElementById('wall');
  const maxX   = player.parentElement.offsetWidth - 42;

  // 1. Horizontal movement
  x += velX;
  if (x >= maxX || x <= 0) velX *= -1;

  // 2. Gravity pulls velY down each frame
  velY += GRAVITY;
  y    -= velY;
  if (y <= GROUND) { y = GROUND; velY = 0; hasJumped = false; }

  player.style.left   = Math.round(x) + 'px';
  player.style.bottom = (28 + Math.round(y)) + 'px';

  // 3. Collision → jump (fires only once per landing)
  if (isColliding(player, wall) && !hasJumped) {
      velY      = JUMP_FORCE;
      hasJumped = true;
  }

  id = requestAnimationFrame(loop);
}

document.getElementById('btn').onclick = function() {
  running = !running;
  running ? loop() : cancelAnimationFrame(id);
  this.textContent = running ? 'Pause' : 'Start';
};
